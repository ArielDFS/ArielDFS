## Featured Projects

### 🏗️ [Atlas ERP](https://github.com/ariel-dfs/atlas-erp)
Vertical SaaS built in under a week for a land development company. Covers financial management, contracts, and installment tracking — with **Maia**, a floating AI chat assistant powered by Gemini 2.5 Flash, embedded directly into the platform.
`Next.js` `TypeScript` `Supabase` `Prisma` `Tailwind v4` `Vercel`

---

### 📊 [Extrator Vivo GO](https://github.com/ariel-dfs/extrator-vivo-go)
Automated ETL pipeline that extracts telecom sales data from the Vivo GO portal, consolidates it via DuckDB + Parquet, and loads it into BigQuery for Power BI dashboards. Production-grade, runs daily.
`Python` `DuckDB` `Apache Parquet` `BigQuery` `GCP`

---

### 🤖 [LABATENDE](https://github.com/ariel-dfs/labatende)
WhatsApp chatbot for automated customer intake and triage. Built with Evolution API + FastAPI, using Claude API as the AI fallback for complex queries.
`FastAPI` `Evolution API` `Claude API` `Python`
